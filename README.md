# Getting started

Tested on Linux and Windows. Python 3.10

### Install

``` bash
pip3 install -r requirements.txt
```

### Run

On Linux run in the console

``` bash
python3 srv_manager.py 1
```

On Windows change the path on the escape.bat file. Then double click it
